--- ያማርኛ ፊደላት መወጠኛ፡፡ ሀ፤ ሳይሉ ጽሕፈት ውል ሳይዙ ሙግት፡፡ ፪/ የግዕዛን ፊደላት መጀመሪያ፡፡ ሀ፤ ባሉ ደሞዝ በሉ፡፡ ፫/ ሀብት = ሀብት፡፡ ፬/ ስሙ = ሃሌታው ሀ፡፡ ሀብት ውሂብ ሃይማኖት ቡሄ ወህኒ ሆይ ---
የሀ ተከታይ፡፡ ፪/ መስተዋድድ፡፡ ለወገኑ ሰጠ፡፡ ለአባቱ ላከ፡፡ ለቤት ሣንቃ ለሰው አለቃ፡፡ ለራስ ምታት ጩኽበት ለሆድ ቍርጠት ብላበት፡፡ ለሚስት ያጐርሰዋል ለተመታ ይክሰዋል፡፡ ፪/ በአናቅጽ አርእስት ላይ እየተጫነ፤ አድራጊነትን አስደራጊነትን፤ ተደራጊነትን፤ ተደራራጊነት፤ አደራራጊነትን፤ አዳራጊነትን፤ ተደራጊነት፤ ይገልጻል፡፡ ለመሥራት የሚደፍር መብላትን አይደፍርም፡፡ ለማስገደል ነው፡፡ ለመገደል ነው፡፡ ለመገዳደል ነው፡፡ ለማገዳደል ነው፡፡ ለማጋደል ነው፡፡ ለመጋደል ነው፡፡ ፫/የበጎ ማድረጊያ፡፡ ፈቀደለት፡፡ ፈረደለት፡፡ ፭/ ለብሰ = ለበሰ ለየ ሉድ ሊባ ላላ ሌባ ልጥ ሎጥ፡፡
ሦስተኛ፡፡ ፪/ የጽሑፍ ምሥጢር አደላዳይ፡፡ ፬/ ሐውልት = ሐውልት አጌ = ሐጌ፡፡ ፭/ ስሙ = ሐመሩ ሐ፡፡ ሐውልት እሑድ ሒሳብ ሐሳብ እግዚአብሔር ሕዝብ ብሖር
አራተኛ፡፡ ፪/ የአናቅጽ አርእስት መነሻ፡፡ መምጣት፤ መኼድ፡፡ መለመን የለመደ ምላስ በሕልሙ አቁማዳን ይዋስ፡፡ ፫/ መስቀል = መስቀል መጣ ሙዳ ሚና ማማ ሜታ ምሳ ሞራ
አምስተኛ፡፡ ፪/ የጽሑፍ ምሥጢር አደላዳይ፡፡ ሴት፡፡ ሤት፡፡ ፬/ ሠርሐ = ሠራ፡፡ ፭/ ስሙ ንጉሡ ሠ፡፡ ሠራ ንጉሠ መሢሕ ሣለ ሤት ሥራ ሦስት 
ሥድስተኛ፡፡ ፪/ ረብሐ = ረባ፡፡ ረባ ሩብ ሪቅ ራራ ሬት ርግብ ሮሮ
ሰባተኛ፡፡ ፪/ የጽሑፍ ምሥጢር አደላዳይ፡፡ ሳለ፡፡ ሣለ፡፡ ፫/ ሰማይ = ሰማይ ተዝካር = ተስካር፡፡ ፬/ ስሙ = እሳቱ ሰ፡፡ ሰባ ሱቲ ሲራክ ሳተ ሴት ስስ ሶራ
ስምንተኛ፡፡ ፪/ መድረሻቸው ሠ፤ ሰ ለኾኑ አናቅጽ ቅጽልን መወሰኛ፡፡ ነገሠ - ነጋሽ፤ መናገሻ፤ መንገሻ፡፡ ካሳ - ከሽ፤ መክሻ፡፡ ፫/ ሢመት = ሹመት፡፡ ሲበት = ሽበት፡፡ መስየ = መሸ፡፡ ሸተ ሹም ሺሕ ሻጣ ጋሼ ቀዳሽ ሙሾ
ዘጠነኛ፡፡ ፪/ ቀብዐ = ቀባ ቀባ ቁና ቂጣ ቃታ ቄራ ቅል ቆዳ
ዓሥረኛ፡፡ ፪/ መስተዋድድ፡፡ (ሀ) የማድረጊያ፡፡ በዱላ ቀነደበው፡፡ (ለ) የቦታ፡፡ በተራራው ላይ አለፈ (ሐ) የጊዜ፡፡ በሥድስት ሰዓት ይመጣል፡፡ (መ) አመዛዛኝ፡፡ ባለፈ ነገር አትቈጭ፡፡ ፫/ ያናቅጽ ባዕድ መነሻ (ለጊዜያት)፡፡ በተሰበረ፡፡ በተሰበረ ነበር፡፡ በመጣ ኖሮ፡፡ ፬/ ስለ፡፡ በእገሌ በሞቱ፡፡ ፭/ በልዐ = በላ፡፡ ዘንመ = ዘነበ፡፡ በመድሎተ ልቡ = በልቡ ሚዛን፡፡ በላ ብቡ ቢያ ባላ ቤት ብዙ ቦይ፡፡
ዓሥራ አንደኛ፡፡ ፪/ የተደራጊና የተደራራጊ የተደራጊም፤ መመደቢያ፡፡ ተገደለ፡፡ ተገዳደለ፡፡ ተጋደለ፡፡ ፫/ከ፡፡ ተኩል = ከእኩል፡፡ ፬/ ተከለ = ተከለ፡፡ ተላ ቱር አላቲ ታታ ስታቴ ትል መቶ፡፡
ዓሥራ ኹለተኛ፡፡ ፪/ መድረሻቸው፤ ተ፤ ለኾኑ፤ አናቅጽ፤ ቅጽልን፡፡ መወሰኛ፡፡ ዛተ - ዛች፤ መዛቻ፤ ዛቻ፡፡ ፫/ በልኣት = በላች፡፡ ኄር = ቸር፡፡ ክህለ = ቻለ፡፡ ቸከ ምቹ ቡቺ ቻለ ልቼ ችኮ ቀበርቾ፡፡
ዓሥራ ሦስተኛ፡፡ ፪/ የጽሑፍ ምሥጢር አደላዳይ፡፡ ድኅነት፡፡ ድኽነት፡፡ ፫/ ኅብል = ኅብል፡፡ ፬/ ስሙ = ብዙኃኑ ኀ፡፡ ኅብል ድኁር ኂሩት ኃይል ርኅራኄ ኅዳር ኆኅት፡፡
ዓሥራ አራተኛ፡፡ ፪/ ነገር = ነገር፡፡ ዋሕድ = ኣንድ፡፡ ነጋ ኑግ ወህኒ ወህኒ ናዳ ብያኔ ንጉሥ ኖረ
